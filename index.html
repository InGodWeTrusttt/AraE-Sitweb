Installation Windows

Objectif : clair et minimal, compatible avec une distribution via GitHub Releases.

Étapes

  1. Télécharger l’installeur (Release GitHub recommandée).
  2. Exécuter le fichier (.exe / .msi).
  3. Lancer AraE.

Distribution (recommandée)

GitHub Releases garantit un versioning clair, un changelog, et un lien “latest” stable. Quand l’installeur existe, le bouton principal pointera directement vers la dernière release.

FAQ

Réponses aux questions qui bloquent l’adoption.

Session vs Monde : quelle différence concrète ?
Session = versionnée et chronologique (instant T). Monde = référentiel consolidé + comparaison des versions.
Où sont stockées les données ?
Stockage local (offline-first). La synchronisation multi-machines peut venir ensuite selon la roadmap.
Pourquoi Windows d’abord ?
Priorité à une première distribution stable et une boucle de feedback courte avant extension à d’autres OS.
